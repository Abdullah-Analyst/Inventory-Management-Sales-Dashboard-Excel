# Inventory Management & Sales Performance Dashboard

## 📌 Project Overview

This project is an interactive **Inventory Management & Sales Performance Dashboard** developed in **Microsoft Excel**. The dashboard provides a comprehensive view of inventory and sales data, enabling users to monitor business performance through dynamic visualizations and KPI tracking.

Using Pivot Tables, Pivot Charts, Slicers, and Dashboard Design techniques, the solution transforms raw inventory data into actionable business insights.

---

## 🎯 Objectives

The dashboard was designed to:

- Monitor overall sales performance
- Track profitability across product categories
- Analyze warehouse performance
- Identify top-performing products
- Visualize monthly sales trends
- Monitor order status distribution
- Enable interactive filtering and exploration

## 📈 Dashboard Features

### 1. Sales Value by Category
Visual comparison of sales generated by each product category.

Categories include:

- Storage
- RAM
- CPU
- Video Card
- Mother Board

### 2. Monthly Sales Trend
Line chart displaying monthly sales performance throughout the year.

### 3. Warehouse Performance
Analysis of sales value generated by each warehouse location.

Warehouses include:

- Beijing
- Bombay
- Mexico City
- New Jersy
- San Francisco
- Seattle Washington
- Southlake Texas
- Sydney
- Toronto

### 4. Profit by Category
Comparison of profitability across product categories.

### 5. Order Status Distribution
Breakdown of order statuses:

- Shipped
- Pending
- Canceled

### 6. Interactive Slicers
Users can dynamically filter dashboard results using:

- Warehouse Name
- Category Name
- Order Status

## 🔍 Business Insights

### Category Analysis

- Storage generated the highest sales value.
- CPU generated the highest total profit.
- Mother Board generated the lowest sales value.

### Warehouse Analysis

- Mexico City recorded the highest sales performance.
- Toronto recorded the lowest sales performance.

### Sales Trend Analysis

- February achieved the highest monthly sales.
- July recorded the lowest monthly sales.

### Order Status Analysis

- Shipped orders represented the largest portion of total orders.
- Pending and canceled orders indicate opportunities for process improvement.

## 🚀 How to Use

1. Download the Excel workbook.

4. Use slicers to filter by:
   - Warehouse
   - Category
   - Order Status
5. Explore insights and trends interactively.
